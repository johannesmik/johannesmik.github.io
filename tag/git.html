Often used commands

Tue 07 July 2015 by Johannes Mikulasch in category Technical.

tags: #git #bash 

In this post I'll list all my often used commands, that I tend to forget from time to time.

In Bash

To find 'surface' in all pdf's in a directory

$ find . -name '*.pdf' -exec sh -c 'pdftotext "{}" - | grep --with-filename --label="{}" --color "truncated surface"' \;

In Git

This one let's me interactively add or skip lines of a file to a git commit.

$ git add -p

In Vim

Substitute foo with bar in all lines (%s), but ask for confirmation (c):

:%s/foo/bar/gc

To paste from the system clipboard:

"*p
