Often used commands

Di 07 Juli 2015 by Johannes Mikulasch in category Technical.

tags: #git #bash 

In this post I'll list all my often used commands, that I tend to forget from time to time.

In Bash

To find 'surface' in all pdf's in a directory

$ find . -name '*.pdf' -exec sh -c 'pdftotext "{}" - | grep --with-filename --label="{}" --color "truncated surface"' \;

Spell correction:

$ aspell check textfile.md

Renaming a lot of files. We make use of bash's variable substitution ${var/Pattern/Replacement}. For example like this:

$ for i in *.txt; do mv "${i}" "{i/.txt/_newtext.txt}"; done

In Git

This one let's me interactively add or skip lines of a file to a git commit.

$ git add -p

Sometimes I spot a small typo or whitespace error just after committing, then I can amend the current commit. This should be done before pushing to a remote branch.

$ git add errorfile
$ git commit --amend

In Vim

Substitute foo with bar in all lines (%s), but ask for confirmation (c):

:%s/foo/bar/gc

To paste from the system clipboard:

"*p
