Rubify

So 17 August 2014 by Johannes Mikulasch in category Technical.

tags: #pelican #tests 

Rubify is a small plugin for the pelican blog, that enables Furigana support.

Technical

I use the XHTML Ruby Tags to display furigana. Chrome does support this (standard) notation, but unfortunately Firefox does not... Have to think about another solution in the future maybe.

I use the reddit furigana syntax to markup my Kanjis with furigana.

Examples

The pattern [漢字] (#fg かんじ) will be printed like 漢字(かんじ)

The "plugin" code

Some lines of Python... Use regular expressions to search for the pattern [*](#fg *) and replace it accordingly

from pelican import signals
import re

def rubify(sender):
    if sender._content:
        print "%s rubifying..." % sender
        content = sender._content

        pattern = r'\[([\w]*)\]\(#fg ([\w]*)\)'
        replacement = r'<ruby><rb>\1</rb><rp>(</rp><rt>\2</rt><rp>)</rp>'

        sender._content = re.sub(pattern, replacement, content, flags=re.UNICODE)

def register():
    signals.content_object_init.connect(rubify)

Simple, isn't it :)
